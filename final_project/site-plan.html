Daycare Connect - Site Plan

Site Name

Daycare Connect

This name was selected because it represents a platform where parents can stay connected with their children's daycare, monitor activities, meals, and staff knowledge, ensuring trust and transparency.

Optional domain availability: daycareconnect.com

Site Purpose

The website provides parents with a comprehensive overview of multiple daycare locations. Parents can:

  • Check daily meals and nutrition information.
  • Review employee qualifications and skills.
  • Locate daycare centers and access contact information.
  • Receive updates on childrenâ€™s daily activities and development.

Scenarios

  • Scenario 1: "Which daycare location has certified teachers with experience in early childhood education?"
  • Scenario 2: "What meals are being served this week, and do they meet my child's dietary needs?"

Color Schema

The following colors will be used across the website:

  • Primary Color: Green (used for headings, logo accents, and buttons)
  • Secondary Color: Orange (used for highlights, links, and call-to-action buttons)
  • Background Color: Light gray (used for page background)
  • Text Color: Dark gray (used for body text)

Typography

  • Roboto: Primary font for headings and body text.
  • Font Weights: Regular 400 for body text, Bold 700 for headings.
  • Optional special font for accents or logo can be added if desired.
